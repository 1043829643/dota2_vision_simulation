下面是一版偏“可落地执行”的 PRD。重点会围绕你新增的方向：多战队眼位横向比较、最佳 / 最差眼位排行、跨队伍视野策略画像、可解释的数据指标体系。

PRD：Dota 2 多战队守卫视野分析与眼位价值比较系统
1. 产品概述
1.1 产品名称

Dota 2 Vision Intelligence Platform

简称：VIP / Vision Lab

1.2 产品定位

面向 Dota 2 职业战队、教练组、数据分析师的视野策略分析工具。

系统基于比赛 replay 解析、Valve 原生 cache.fow 视野遮挡算法、7.41 地图坐标投射与守卫生命周期数据，对 Observer Ward 与 Sentry Ward 的实际价值进行量化，并支持多战队之间的眼位习惯、眼位收益、反眼风险、视野控制能力的横向比较。

一句话定位：

把“这个眼好不好”从经验判断变成可量化、可对比、可复盘、可沉淀的战术数据。

2. 背景与问题
2.1 当前问题

职业战队和分析师在复盘视野时，常见痛点包括：

眼位价值难量化
传统数据只能看到插眼数量、反眼数量，无法判断某个眼是否真正产生了信息价值。
缺少多战队横向比较
一支队伍某个眼位用得多，不代表这个眼位好。需要和其他队伍、强队、同阵营、同时间段进行比较。
缺少“最佳 / 最差眼位”标准
分析师想知道：
哪里的眼看到人最多？
哪里的眼平均存活最久？
哪里的眼容易被反？
哪里的眼看似常用但实际收益低？
哪些高价值眼位本队很少使用？
眼位习惯无法沉淀成战术资产
队伍的眼位偏好、时间点习惯、Roshan 前布控模式、落后局防守眼习惯，通常散落在人工复盘笔记中，难以长期积累。
单场复盘和赛前 scouting 没有打通
单场比赛看到了某个眼，但很难知道：
这是偶然使用，还是该队长期习惯？
这个眼在全职业样本中属于高价值还是低价值？
对手是不是经常在这个时间点插这个眼？
3. 产品目标
3.1 核心目标

建立一个多战队、多比赛、多地图区域的守卫视野价值分析系统，帮助分析师回答以下问题：

哪里是当前版本最高价值眼位？
哪里是低效、容易被反、存活短的差眼位？
不同战队在眼位选择上有什么差异？
本队相比强队，哪些眼位用得太多 / 太少？
对手有哪些固定眼位习惯可以被针对？
某个眼位在不同队伍、不同时间段、不同局势下表现如何？

8. 指标体系设计

这是这个产品最关键的部分。

如果只看“看到人最多”，会误导分析师。因为一个眼看到人少，可能是因为它成功威慑了对手，也可能是因为放在了低价值区域。

所以系统需要同时提供 原始指标、效率指标、风险指标、机会校正指标。

8.1 基础指标
指标	英文名	说明
使用次数	Placement Count	该眼位被放置的次数
使用率	Pick Rate	某队使用该眼位的频率
平均存活时间	Avg Lifetime	从放置到消失 / 被反的平均时间
中位存活时间	Median Lifetime	比平均值更抗极端值
被反率	Deward Rate	被敌方清除的比例
快速被反率	Fast Deward Rate	放下后 N 秒内被反的比例
看到英雄总秒数	Enemy Hero Seen Seconds	敌方英雄处于该眼视野内的总时间
平均看到英雄秒数	Avg Seen Seconds	单个眼平均看到多少秒敌方英雄
看到次数	Sighting Count	连续观察片段数量
首次发现次数	First Contact Count	敌方英雄此前不可见，首次被该眼发现
看到不同英雄数	Unique Heroes Seen	该眼最多覆盖到几名不同敌方英雄
隐身真视秒数	True Sight Seconds	真眼覆盖隐身单位的时间
视野重叠率	Overlap Rate	与己方其他视野重复的比例
8.2 效率指标
指标	计算方式	作用
单分钟视野收益	Seen Seconds / Lifetime Minutes	判断眼活着时是否真的有信息
单次插眼收益	Seen Seconds / Placement	判断平均插一颗眼能产生多少信息
首次发现效率	First Contacts / Placement	判断是否能捕捉敌方关键动向
高价值英雄观察秒数	按英雄经济 / 位置加权	看到 carry 和看到辅助价值不同
关键时间窗口收益	Roshan / 推塔 / 控符前后的信息量	判断眼是否服务战术目标
低重叠收益	Seen Seconds × (1 - Overlap Rate)	惩罚重复覆盖
8.3 风险指标
指标	说明
被反率	该眼位被敌方排掉的概率
平均被反时间	从放置到被反的平均时间
快速被反率	放置后 30 / 60 / 90 秒内被反
放眼者暴露率	放眼过程中，放置英雄是否被敌方看到
放眼后死亡率	放眼后短时间内放置英雄是否死亡
反眼成本	敌方使用多少 Sentry / 时间来清该眼
路线风险	插这个眼的常见路径是否容易被看到
8.4 机会校正指标

这是多战队比较中非常重要的一层。

单纯比较“看到人多少”并不公平。比如：

防守眼在敌人不进攻时可能看不到人。
进攻深眼如果队伍大劣，很难插出去。
Roshan 眼在没有 Roshan 争夺时收益自然低。
某队打得很主动，眼天然看到人更多。

因此要设计 Opportunity Adjusted Metrics。

机会校正可见收益
Opportunity Adjusted Seen Rate =
实际看到敌方英雄秒数 / 理论机会区域内敌方英雄秒数

解释：

如果敌方英雄曾经经过这个眼位附近的潜在影响区域，但实际没被看到，说明这个眼可能被地形、树木、角度限制，或者位置不够好。

机会区域定义

对某个 Ward Spot，定义一个潜在影响区域：

Observer Ward:
以该眼位为中心，根据真实 FOW 模拟得到最大可见区域 + 周边 buffer

Sentry Ward:
1000 半径真视范围 + 普通视野条件判断
机会校正的作用

它能区分：

情况	原始看到秒数	机会校正后判断
敌人根本没来	低	不一定是差眼
敌人路过但没看到	低	可能是差眼
敌人经常经过且多次看到	高	高价值眼
敌人经过但被树 / 高坡挡住	低	位置可能需要微调
8.5 眼位价值评分

系统可以给每个眼位一个综合分，但必须可解释。

推荐评分模型
Ward Value Score =
30% 机会校正看到英雄秒数
+ 20% 首次发现次数
+ 15% 高价值英雄观察秒数
+ 15% 目标区域窗口收益
+ 10% 存活表现
+ 5% 低重叠覆盖
+ 5% 反隐 / 真视收益
- 快速被反惩罚
- 高重叠惩罚
- 低样本置信度惩罚

前端不应该只显示一个分数，而要展示拆解：

Vision Value Score: 84 / 100

贡献来源：
+ 看到敌方核心 42 秒
+ Roshan 前 120 秒内提供关键入口视野
+ 首次发现敌方开雾 2 次
- 与己方河道眼有 28% 重叠
- 被反率高于平均值
9. 最佳 / 最差眼位定义
9.1 “最佳眼位”不止一种

前端需要支持多种排行模式。

排行类型	说明	适合回答
看到人最多	平均看到敌方英雄秒数最高	哪些眼信息量最大
存活最久	平均 / 中位存活时间最高	哪些眼最不容易被反
效率最高	单分钟看到英雄秒数最高	哪些眼活着时最有价值
首次发现最多	First Contact 高	哪些眼最容易发现敌方动向
Roshan 价值最高	Roshan 前后收益高	控盾前该插哪里
反隐价值最高	真视覆盖隐身收益高	打隐身英雄怎么布控
综合评分最高	多指标加权	通用意义上的强眼位
9.2 “最差眼位”也不止一种
排行类型	说明	适合回答
看到人最少	平均看到敌方英雄秒数低	哪些眼信息量差
存活最短	平均存活时间短	哪些眼容易被反
快速被反最多	60 秒内被反率高	哪些眼位太常规
重叠最高	与己方其他眼重复覆盖	哪些眼浪费
机会浪费最大	敌人经过附近但实际没看到	哪些眼位置不准
使用高但收益低	高频低效眼	哪些习惯需要调整
综合评分最低	多指标加权	通用意义上的差眼位

19. 风险与注意事项
19.1 “看到人少”不一定代表差眼

一个防守眼可能没有看到人，是因为它成功阻止了敌方入侵。

解决方式：

引入机会校正指标
显示敌方是否接近过该区域
给出“低机会样本”标签
不只用原始看到秒数判断好坏
19.2 “存活久”不一定代表好眼

有些眼活得久，是因为它放在没人经过的地方。

解决方式：

存活时间必须和看到英雄秒数、目标窗口收益一起看
排行榜区分“存活最长”和“综合价值最高”
19.3 高频眼不一定是好眼

职业队常用的眼位可能只是习惯眼，容易被反。

解决方式：

展示使用率与价值分的二维图
标记“高频低效”
标记“高频高被反”
19.4 多战队比较需要控制样本偏差

不同战队比赛风格不同：

有的队伍节奏快
有的队伍经常领先
有的队伍更偏防守
有的队伍比赛时间更长
有的队伍更常打 Roshan 阵容

解决方式：

按时间段比较
按局势比较
按阵营比较
按 patch 比较
提供 benchmark 置信度
默认显示样本数

23. 最关键的产品判断

这个系统不要把“最佳眼位”定义成单一答案。

应该让用户选择：

我想找：
1. 看到人最多的眼
2. 存活最久的眼
3. 最不容易被反的眼
4. Roshan 前最有价值的眼
5. 打隐身英雄最有价值的真眼
6. 某战队最常用的眼
7. 某战队最容易被预测的眼
8. 本队相比强队最缺少的眼
9. 本队最高频但最低效的眼

这样产品才会真正符合职业分析师的工作方式。

24. 最终产品价值

这版产品上线后，分析师可以得到几类非常实用的结论：

1. 当前版本哪些眼位整体最强
2. 当前版本哪些眼位看似常用但收益很差
3. KRD 相比强队缺少哪些高价值眼位
4. KRD 哪些固定眼位容易被对手针对
5. 对手在哪些时间点喜欢插哪些眼
6. 哪些真眼区域对隐身英雄限制最大
7. Roshan 前哪些眼位最能决定视野控制
8. 哪些眼位存活久但信息价值低
9. 哪些眼位短命但能提供关键首次发现

一句话总结这版 PRD 的核心：

这个产品的重点不是展示“眼在哪里”，而是回答“这个眼在多战队样本里到底值不值、谁用得好、谁用得差、什么时候该用、什么时候容易被针对”。
