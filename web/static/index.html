Dota 2 战队视野分析

从战队开始研究视野习惯：先筛选比赛，再计算眼位实际看到的敌方英雄、真眼覆盖隐身收益，以及每场比赛的秒级地图时间轴。
先搜索战队并选择比赛。计算后这里会显示战队视野概览、数据可信度和核心指标。

比赛样本

0 场，已选 0 场

使用方式

1. 搜索战队：输入战队 Tag，并可按对手、Patch、赛事过滤。
2. 选择样本：先用最近 N 场快速选择，再手动调整。
3. 计算：后端会按所选时间段计算假眼看到的敌方英雄，以及隐身英雄是否被真眼覆盖。
4. 验证：进入单场时间轴，在地图上逐秒检查眼位、可见敌方英雄和假眼 FoW 格。
计算完成后，可以在这里选择某一场比赛播放眼位视野时间轴。
计算完成后，这里会显示英雄可见明细、眼位贡献排行，并支持导出 JSON / CSV。

点位库与排行榜

基于当前已选比赛计算眼位实例、点位聚类、最佳/最差点位和点位详情。这个计算会复用 native FoW、动态树和真眼覆盖判断。
先在“比赛样本”里选择比赛，然后点击“计算点位库”。

多战队眼位对比

从下方已预热的战队里点击头像，选择 2-20 支进行对比（直接复用预热好的点位库）。想加入新战队？请先到「预热计算」把它算好。可用上方阵营切换分别看天辉/夜魇。
加载已预热战队中…
添加至少两支战队后点击“计算对比”。

多战队眼位聚合

从下方已预热的战队里点击头像选择若干支，把它们的眼位叠加到同一张地图上，按密度观察「大家常在哪做眼」。绿色=假眼(Observer)、蓝色=真眼(Sentry)分开呈现。使用顶部时间段设置。
加载已预热战队中…
选择至少一支已预热战队后点击“聚合眼位”。

预热计算

先搜索并添加战队（确认能查到比赛后再加入列表），然后一次性把每支战队的“点位库与排行榜”和这批战队的“多战队眼位对比”都算好写入缓存。预热完成后会持久保存到服务器，其他人打开本页也能看到历史记录并一键打开。使用顶部的时间段设置。
填入战队 Tag，点击“开始预热”。预热完成后可从结果卡片一键打开对应的点位库或对比。
